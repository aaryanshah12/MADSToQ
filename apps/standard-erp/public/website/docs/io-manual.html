Inward-Outward Portal — User manual

Material movement, quotations, and master data for chemical operations.

Inventory Inward-Outward PMC Personal Sales

1. Sign in

  1. Open /inward-outward/login (or your deployed I/O URL).
  2. Pick the factory from the sidebar switcher when you have access to more than one site.

2. Daily operations

  • Dashboard — Summary of recent inward/outward activity.
  • Inward — Record material received; link products, quantities, and documents.
  • Outward — Dispatch and outward listings.
  • Domestic / International — Region-specific customer and shipment flows.
  • Quotation — Create and track quotations before outward confirmation.

3. Master (tabbed menu)

All setup data lives on one page with tabs (same pattern as PMC Master):

  • Products — SKU catalog; supports CSV import.
  • Companies — Suppliers and customers.
  • Units — Measurement units for line items.
  • Locations — Countries, states, and cities.
  • PDFs — Label, letterhead, and customer print templates.
  • Numbering — Document number sequences.

4. Typical workflow

  1. Configure Master data (products, companies, units) before transactional entry.
  2. Create inward entries when material arrives.
  3. Use quotations for customer pricing, then outward when goods leave.
  4. Print address or label PDFs from configured templates where enabled.

5. Tips

  • Keep company records accurate — domestic and international screens reuse master addresses.
  • Review numbering settings before go-live to avoid duplicate document IDs.
