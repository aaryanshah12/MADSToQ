PMC Portal — User manual

Product pricing with reference-controlled raw material cost (RMC).

1. Sign in

  1. Open /pmc and sign in with the email and password provided by your administrator.
  2. After login, the app loads master data (raw materials, products, references) into memory for fast editing.

2. Recommended setup order

  1. Master → Raw materials — Add every material used in recipes (name + unit, e.g. Kg).
  2. Master → Products — Create each finished product, then open Edit recipe to set BOM lines (quantity per material, mark one as primary).
  3. Reference Number — Save a new reference when raw-material prices change; the system assigns a reference ID.
  4. RMC — For each product, enter overhead, batch multiplier, and yield per reference to calculate final RMC (see the RMC menu in the sidebar).

3. Dashboard

Summary counts and shortcuts to raw materials, references, and recent products. Use it for a quick health check before updating prices.

4. Reference Number

  • Create a reference when you need a new price list for raw materials.
  • Enter each material rate; saving creates a durable reference you can reuse on product pricing sheets.
  • Past references remain in history for audit and comparison.

5. RMC (operational pricing)

  • This section is for RMC calculation, not for defining BOM (that is under Master → Products).
  • Select a product, pick a reference, and fill overhead / yield / tons per kg.
  • The sheet shows calculated RMC based on recipe quantities and reference prices.

6. Master (tabbed menu)

Like other MADSToQ portals, Master uses a horizontal menu:

  • Raw materials — Add, view, edit, or deactivate materials. Inactive materials stay in history but drop from pick lists.
  • Products — Add products and open the recipe editor to link materials and quantities. New products open the recipe screen automatically.

7. Recipe editor

  1. From Master → Products, use the edit action on a product.
  2. Add rows for each raw material and quantity; mark exactly one row as primary where required.
  3. Save the recipe before pricing the product under a new reference.

8. Tips

  • Deactivate instead of deleting when a material or product is retired — pricing history stays intact.
  • If RMC looks wrong, confirm the active reference, recipe quantities, and primary material flag.
  • Use User manual in the sidebar anytime to reopen this guide.
